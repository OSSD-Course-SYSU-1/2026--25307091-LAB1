## 🤖 DeepSeek AI 临时管理员系统（2026-06-23 新增）

### 核心设计
- **会话级临时授权**：游戏开始时激活管理员会话，对局结束全量自动复原
- **问答式交互**：AI 根据游戏状态出题 → 玩家选择 → 执行 → 恢复游戏
- **快照-复原机制**：所有管理员修改记录到 `modifications` 日志，`handleGameOver` 时逆序复原

### 授权命令集
| 命令 | 功能 | 会话结束后 |
|------|------|-----------|
| `spawn_food` | 生成普通/有毒食物 | 复原食物数量 |
| `set_speed` | 调整游戏速度 | 恢复原始速度 |
| `grow_snake` | 蛇身 +N 节 | 裁剪回原始长度 |
| `shrink_snake` | 蛇身 -N 节 | 裁剪回原始长度 |
| `toggle_invincible` | 无敌模式（穿墙环绕） | 自动关闭 |
| `get_stats` | 查询状态（只读） | 无需复原 |
| `broadcast` | 画布广播消息 | 无需复原 |

### 架构层次
- **ArkTS 原生层**：`DeepSeekService.ets` — DeepSeek Chat API 封装，API Key 仅存原生层
- **JSBridge 桥接**：`WebViewPage.ets` — `callDeepSeek()` 异步转发 + `runJavaScript` 回调
- **H5 游戏层**：`index.js` — `AdminSession` 快照模块 + `AdminCommands` 执行器 + 问答弹窗逻辑

### 交互界面
- 右下角 🤖 浮动按钮（游戏开始后显示）
- 点击后游戏暂停，弹窗显示加载→AI 问题+选项
- 玩家选择后执行命令→弹窗关闭→游戏恢复
- UI 风格与暂停弹窗一致（深色主题 + 蓝色边框）

### 降级策略
- DeepSeek API 不可用时自动使用预置降级问答（`buildFallbackQuestion`）
- 模拟器环境下 JSBridge 不可用时降级为纯前端问答

### 安全
- API Key 仅存在于 ArkTS 层，不暴露到 H5
- 管理员不可修改 `score` / `highScore`（硬编码禁止）
- `.gitignore` 排除 `DeepSeekService.ets`

## 🍎 双食物系统

- **普通食物**（50%概率）：使用原有 food1.png / food2.png 图片渲染
- **有毒食物**（50%概率）：Canvas 像素风绿色毒蘑菇小怪，带绿色毒气光晕
  - 深绿伞盖 + 荧光绿斑点 + 发光大眼 + 浅绿菌柄 + 锯齿嘴
- 普通食物：蛇身 +1，加分递增
- 有毒食物：蛇身缩短，扣分递增，红色 `-xxx` 浮字动画

---

## 📈 迭代等级系统

- 积分+时间双驱动：`level = floor(score/500) + floor(秒/45)`
- 每级普通食物奖励 +50，毒惩罚 = 普通奖励 × 50%
- 毒食物缩短节数：`2 + floor(level/2)`
- Lv.0~Lv.8 动态变化

---

## 🧠 自学习 AI

- 每 5 局自动分析平均分+平均时长
- 玩得好 → 提高分阶、加速时间步、增加毒概率
- 玩得差 → 降低分阶、延长时间步、减少毒概率
- 参数持久化至 `localStorage`（`snakeLearning`）
- 4 个动态参数：分阶(200~800)、时间步(20~55s)、刷新加速(0.8~2.5)、毒概率(30%~70%)
